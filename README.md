## Hướng dẫn chạy Notebook 🤓
**Với file EDA:**
+ Nếu chạy offline thì comment out cell mount Google Drive
+ Thay đổi DATA_DIR thành đường dẫn vào dataset

**Với file Huấn luyện model:**
_(datathon-prediction-final.ipynb)_
+ Thay đổi DATA_DIR thành đường dẫn vào dataset (config mặc định chạy trên kaggle competition)
+ Thay đổi OUTPUT_FILE thành đường dẫn lưu file kết quả bạn muốn (để nguyên nếu lưu vào folder hiện tại)
